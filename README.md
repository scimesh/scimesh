# To use Activiz library:
Visual Studio -> Solution Explorer -> Assembly-CSharp -> Add -> Existing Item:

# Unity to Android building  
Player Settings -> Andorid -> Other Settings -> Configuration -> API Compatibility Level -> .NET 2.0
